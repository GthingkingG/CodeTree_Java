# CodeTree_Java

CodeTree 플랫폼 Java 문제 풀이 학습.

## 폴더 구조
날짜별 폴더(`YYMMDD/`)에 그날 풀이한 문제들을 저장.

```
250313/   # 2025-03-13 풀이
└── ...
```

## 실행
```bash
cd 250313
javac <File>.java
java <ClassName>
```
